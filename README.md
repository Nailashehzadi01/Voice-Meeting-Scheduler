### 🔧 **Step-by-Step Installation**

</div>

<details>
<summary><strong>📥 1. Clone Repository</strong></summary>

```bash
git clone https://github.com/Nailashehzadi01/Voice-Meeting-Scheduler.git
cd voiceschedule-pro
```

</details>

<details>
<summary><strong>📦 2. Install Dependencies</strong></summary>

```bash
npm install
```

**Dependencies installed:**
- Express.js server framework
- Chart.js for analytics visualization  
- Font Awesome icons library
- TypeScript development tools

</details>

<details>
<summary><strong>🚀 3. Start Development Server</strong></summary>

```bash
npm run dev
```

**Server starts on:** `http://localhost:8080`
**Features enabled:** Hot reload, TypeScript compilation, Express routing

</details>

<details>
<summary><strong>🎤 4. Enable Voice Features</strong></summary>

1. **Grant Microphone Permission** when prompted
2. **Test Voice Recognition** using the microphone button
3. **Try Sample Commands**:
   - "Schedule meeting with team tomorrow at 10 AM"
   - "Book call with client on Friday at 2 PM"

</details>

## 🚨 Troubleshooting

### Common Issues
1. **Voice Recognition Not Working**
   - Ensure microphone permissions are granted
   - Check browser compatibility (Chrome, Edge recommended)
   - Verify microphone hardware functionality

2. **Charts Not Displaying**
   - Ensure JavaScript is enabled
   - Check for browser console errors
   - Verify Chart.js library loading

3. **Data Not Saving**
   - Check browser storage permissions
   - Ensure sufficient storage space
   - Clear browser cache if needed

### Browser Compatibility
- **Recommended**: Chrome 80+, Edge 80+, Firefox 75+
- **Voice Features**: Chrome and Edge (WebKit browsers)
- **Mobile**: iOS Safari 14+, Android Chrome 80+
